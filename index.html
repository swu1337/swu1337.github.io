Imperium Website

HTML, CSS, JavaScript and jQuery

Imperium website was a school project focused on the designing and realise phase. First, we focused on getting all the information from the client. Secondly, we converged the given information into a debriefing, user stories and requirements. Thirdly, we used the stories and requirements to make a paper prototype. We used this prototype to test the concept and ask for improvements. After prototyping we made a sitemap and navigation model to study/set the structure of the website. After that we made clickable prototypes to test the interaction and user experience. We used heuristic evaluation and scenario based testing to achieve that. Style guide is the next step, in the style guide we specify guidelines for the website. After that is we had to realise the website using only HTML, CSS and JavaScript/jQuery. Bootstrap was not allowed so I had to style everthing myself. HTML, CSS, JS, jQuery was used in this project.
