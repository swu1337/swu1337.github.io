Imperium Website

HTML, CSS, JavaScript and jQuery

Imperium Website is school project focused on implementing the designing and realise phase into practice. First we focus on getting all the information from the client. We converge information into a debriefing, user stories and requirements. The next step is to convert those information into a prototype(paper). With this prototype we can ask for feedback. After prototyping we made a sitemap and navigation model to study the structure of the website. After that we made clickable prototypes to test the interaction and user experience. Style guide is the next step, in the style guide we specify the style guidelines for the website. After that is realising the website. Bootstrap was not allowed so I had to style everthing by myself. HTML, CSS, JS, jQuery was used in this project.
